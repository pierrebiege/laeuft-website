## Design-Entscheidungen

- **Layout**: Horizontal Scroll Cards (Apple-Style Carousel)
- **Modal-Design**: Minimalistisch
  - Grosser Titel (bis 7xl)
  - Ein Satz Beschreibung
  - Bilder im Fokus
  - Tech Stack ganz unten, dezent (25% Opacity)
  - Features entfernt (Steve Jobs Analyse: Kunden interessiert nicht die Technik)

Einfuhrung

Über das Brigerbad Run & Trail Event

Unsere Vision

Der Brigerbad Run & Trail ist ein Lauf- und Trailrunning-Event im wunderschönen Walliser Brigerbad. Dieses Corporate Identity Manual definiert die visuelle Identität und stellt sicher, dass alle Kommunikationsmittel einheitlich und professionell auftreten.

Laufen verbindet Menschen mit der Natur und sich selbst. Der Brigerbad Run & Trail schafft ein unvergessliches Erlebnis in der spektakulären Bergwelt des Wallis - für alle Leistungsstufen.

Bitte verwende die bereitgestellten Assets und halte dich an die Richtlinien in diesem Manual, um eine konsistente Markenwahrnehmung zu gewährleisten.

Logo-Schutzzone

Freiraum und Mindestgrössen für optimale Lesbarkeit

X X X X

Mindestabstand X = Höhe des "B" im Schriftzug

Warum eine Schutzzone?

Die Schutzzone stellt sicher, dass das Logo genügend Freiraum hat und nicht von anderen Elementen bedrängt wird. Dies garantiert maximale Wirkung und Lesbarkeit in allen Anwendungen.

Mindestgrössen

Print (Kombination) min. 35mm Breite
Print (Bildmarke) min. 15mm Breite
Digital (Kombination) min. 150px Breite
Digital (Bildmarke) min. 48px Breite
Favicon 32 x 32px

Logo-Schrift

FF Dax

Die FF Dax ist eine lizenzierte Schrift von FontFont/Monotype. Sie wird ausschliesslich für das Logo verwendet.

Warum FF Dax fürs Logo?

  • Charakteristisch sportlicher Look
  • Abgerundete, freundliche Formen
  • Hohe Wiedererkennung
  • Geschützte Einzigartigkeit der Marke

Das Logo darf nicht mit anderen Schriften nachgebaut werden. Verwende immer die bereitgestellten Logo-Dateien.

Anwendungen

Korrekte und inkorrekte Verwendung des Logos

Richtig
  • ✓ Farblogo auf weissem oder hellem Hintergrund verwenden
  • ✓ Weisses Logo auf dunklem Hintergrund (Navy empfohlen)
  • ✓ Schwarzes Logo fürEinfarbdruck / Stempel
  • ✓ Genugend Freiraum (Schutzzone) einhalten
  • ✓ Originalproportionen immer beibehalten
  • ✓ Nur bereitgestellte Dateien verwenden
Falsch
  • ✕ Logo verzerren, strecken oder stauchen
  • ✕ Logo rotieren oder kippen
  • ✕ Farben des Logos ändern
  • ✕ Schatten, Glanz oder andere Effekte hinzufügen
  • ✕ Logo auf unruhigem oder farbigem Hintergrund platzieren
  • ✕ Logo mit anderer Schrift nachbauen
